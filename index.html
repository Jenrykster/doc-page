First step

So you got your own jenryk bot ? Let's give it a little test to see if everything is working correctly.

jenryk.start() -> dance

Did your robot dance a cool lil dance ? If your answer is a "YES !" then your robot is working perfectly. If not please refer to this chapter .

Jenryk features

What can jenryk do

  • Dance
  • Code websites
  • Laugh at cat memes
  • Talk about himself as if he was a robot

What can't jenryk do

  • Sing
  • Code cool websites
  • Draw
  • Listen to bad music

Basic API

Dance(break, robot-dance, ankward-dance)

jenryk -> dance(option)

Code website(tribute, survey, landing, documentation)

jenryk -> code(option, askForFeedback=true)

Laugh at cat memes

!Does this automatically, no need for code

You can create your own function like this:

jenryk -> codeDance -> code AND dance
